How To Use

Step 1: Open main.exe (or main) using your terminal
Step 2: The Interactive Application for Hospital Management will open up. Continue working with it.

How To Run main.exe (or main) Using Terminal

Step 1: Open the folder containing main.exe (or main) in File Explorer
Step 2: In any empty space in the File Explorer, right click holding the shift key. The explorer pop up will open up.
Step 3: In the explorer pop up, search for "Open Powershell Window here" and click on it.
Step 4: The Powershell window will open.
Step 5: In the powershell prompt run the following command:
> ./main.exe
If the file is named as main, run:
> ./main
